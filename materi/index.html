Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan mendesain halaman web. HTML terdiri dari serangkaian elemen yang digunakan untuk mengatur struktur konten, seperti teks, gambar, tautan, dan lainnya. Setiap elemen HTML diwakili oleh tag, seperti <p> untuk paragraf, <h1> untuk judul, dan <a> untuk tautan. HTML bekerja bersama dengan CSS (Cascading Style Sheets) dan JavaScript untuk menciptakan pengalaman web yang interaktif dan menarik.

Sejarah HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991 sebagai bagian dari proyek World Wide Web (WWW). Versi pertama HTML sangat sederhana dan hanya mendukung elemen dasar seperti judul, paragraf, dan tautan. Seiring waktu, HTML terus berkembang dengan versi baru seperti HTML 2.0, HTML 3.2, HTML 4.01, dan akhirnya HTML5 yang dirilis pada tahun 2014. HTML5 memperkenalkan banyak fitur baru seperti elemen semantik, dukungan multimedia, dan API yang memungkinkan pengembangan aplikasi web yang lebih kompleks.

Kode Dasar HTML

Setiap dokumen HTML dimulai dengan deklarasi <!DOCTYPE html> yang memberitahu browser bahwa dokumen ini menggunakan HTML5. Struktur dasar HTML terdiri dari elemen <html>, <head>, dan <body>. Elemen <head> berisi metadata seperti judul halaman dan tautan ke file CSS, sedangkan elemen <body> berisi konten yang ditampilkan di halaman web. 

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
</body>
</html>

Penjelasan:

<!DOCTYPE html> : Ini berfungsi Untuk memberitahu browser bahwa dokumen ini menggunakan HTML5.

<html> </html> : Ini berfungsi untuk menandai awal dan akhir dokumen HTML.

<head> </head> : Ini berfungsi untuk sebagai wadah untuk metadata dan elemen-elemen yang tidak langsung ditampilkan pada halaman web contoh nya adalah : <title> DLL.

<body> </body> HTML memiliki fungsi utama sebagai wadah untuk semua konten yang akan ditampilkan pada halaman web.

Elemen HTML

Elemen HTML adalah blok bangunan dasar dari sebuah halaman web. Setiap elemen terdiri dari tag pembuka, konten, dan tag penutup. Contoh elemen HTML termasuk <p> untuk paragraf, <h1> hingga <h6> untuk judul, dan <a> untuk tautan. Beberapa elemen tidak memiliki tag penutup, seperti <img> untuk gambar dan <br> untuk baris baru. 

<h1>Ini adalah judul</h1>
<h2>Ini adalah judul</h2> 
<h3>Ini adalah judul</h3> 
<h4>Ini adalah judul</h4> 
<h5>Ini adalah judul</h5> 
<h6>Ini adalah judul</h6> 
 
<p>Ini adalah paragraf</p>

Atribut HTML

Atribut HTML digunakan untuk memberikan informasi tambahan tentang elemen. Atribut selalu ditulis dalam tag pembuka dan biasanya terdiri dari nama dan nilai. Contoh atribut termasuk href untuk menentukan URL tautan, src untuk menentukan sumber gambar, dan class untuk menentukan kelas CSS. 

<a href="https://wa.me/6288269059314" target="_blank">Klik di sini</a> 
<img src="logo.png">

Formulir HTML

Formulir HTML digunakan untuk mengumpulkan input dari pengguna. Elemen <form> digunakan untuk membuat formulir, dan elemen seperti <input>, <textarea>, dan <select> digunakan untuk membuat berbagai jenis input. Formulir dapat dikirim ke server menggunakan metode GET atau POST. 

<form>
    Nama: <input type="text" name="nama">
    <input type="submit" value="Kirim">
</form> 

<form>
Kelamin: <input type="radio" name="Kelamin">
<input type="submit" value="Kirim">
</form>

<form>
Kelamin: <input type="checkbox" name="Kelamin">
<input type="submit" value="Kirim">
</form>

<form>
Tanggal: <input type="date" name="Kelamin">
<input type="submit" value="Kirim">
</form>

Tabel HTML

Tabel HTML digunakan untuk menampilkan data dalam format baris dan kolom. Elemen <table> digunakan untuk membuat tabel, <tr> untuk baris, <th> untuk header kolom, dan <td> untuk sel data. Atribut seperti border dan colspan dapat digunakan untuk mengatur tampilan tabel. 

<table border="1">
    <tr>
        <th>Nama</th>
        <th>Usia</th>
    </tr>
    <tr>
        <td>Arief</td>
        <td>16</td>
    </tr>
</table>

Media HTML

HTML mendukung berbagai jenis media seperti gambar, audio, dan video. Elemen <img> digunakan untuk menampilkan gambar, <audio> untuk audio, dan <video> untuk video. Atribut seperti src digunakan untuk menentukan sumber media, dan controls digunakan untuk menampilkan kontrol pemutaran. 

<img src="logo.png" alt="Deskripsi">

List HTML

List HTML digunakan untuk menampilkan daftar item. Ada dua jenis list utama: unordered list (<ul>) yang menggunakan bullet points, dan ordered list (<ol>) yang menggunakan angka atau huruf. Setiap item dalam list ditandai dengan elemen <li>. 

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Elemen Semantik

Elemen semantik adalah elemen HTML yang memiliki makna khusus dan membantu mesin pencari dan pengembang memahami struktur halaman web. Contoh elemen semantik termasuk <header>, <footer>, <article>, dan <section>. Elemen ini membuat kode lebih mudah dibaca dan dirawat. 

<article>Ini adalah artikel</article>

Iframe

Iframe digunakan untuk menampilkan halaman web atau dokumen lain di dalam halaman web saat ini. Elemen <iframe> memungkinkan Anda menyematkan konten dari sumber eksternal, seperti video YouTube atau peta Google. 

<iframe src="https://jagongoding.com/web/html/dasar/tipe-element-input/"></iframe>

Meta Tag

Meta tag digunakan untuk menyediakan metadata tentang dokumen HTML, seperti deskripsi halaman, kata kunci, dan penulis. Meta tag ditempatkan di dalam elemen <head> dan tidak ditampilkan di halaman web, tetapi berguna untuk SEO (Search Engine Optimization). 

<meta name="description" content="Belajar HTML">

SVG

SVG (Scalable Vector Graphics) adalah format gambar berbasis XML yang digunakan untuk membuat grafik vektor. SVG dapat diskalakan tanpa kehilangan kualitas dan sering digunakan untuk ikon, logo, dan ilustrasi. 

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>


<svg width="100" height="100">
<rect x="10" y="10" width= "80" height= "80" stroke="black" stroke-width="3" fill="blue">
</svg>

Canvas

Elemen <canvas> digunakan untuk menggambar grafik secara dinamis menggunakan JavaScript. Canvas sering digunakan untuk membuat animasi, grafik interaktif, dan visualisasi data. 

<canvas id="myCanvas" width="200" height="100"></canvas>

Data Attributes

Data attributes adalah atribut khusus yang dimulai dengan data- dan digunakan untuk menyimpan informasi tambahan dalam elemen HTML. Data attributes dapat diakses menggunakan JavaScript untuk membuat halaman web yang lebih dinamis. 

<div data-info="contoh">Elemen dengan Data Attributes</div>

Desain Responsif

Desain responsif adalah pendekatan desain web yang memastikan halaman web terlihat baik di semua perangkat, termasuk desktop, tablet, dan ponsel. Meta tag <meta name="viewport"> dan CSS media queries digunakan untuk mencapai desain responsif. 

<meta name="viewport" content="width=device-width, initial-scale=1">
